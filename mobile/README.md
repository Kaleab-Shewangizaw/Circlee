# Circlee Mobile

This folder contains the Flutter app structure for Circlee.

Flutter is not installed in the current environment, so this repository includes a production-ready directory layout and representative code files rather than a generated Flutter project skeleton.

Recommended bootstrap command on a machine with Flutter installed:

```bash
flutter create mobile_app --platforms=android,ios
```

Then copy the `lib/` structure from this directory into the generated app and wire assets, fonts, and build configuration.
